# Perfect Cut

Try to cut things exactly in half. Inspired by [Halbierung](https://play.date/games/halbierung/).

**Play:** https://yayitswei.github.io/perfect-cut/

- **Mouse:** scroll wheel moves the shape (right-click to switch to rotate), click to cut.
- **Logitech MX Creative Console dialpad:** switch to 🎛 dial mode — big dial moves,
  roller rotates, no Logitech software needed.

Dev extras: [input-lab.html](input-lab.html) (raw input inspector) and
[hidpp-lab.html](hidpp-lab.html) (WebHID HID++ explorer for the dialpad).
